# Visual Documentation | Vinamra Jain™

This document provides a descriptive overview of the key visual interfaces and the design language implemented in the **CRM & Sales Management** platform.

## 1. Design Language & Visual Identity
*   **Branding**: Centered around the **Vinamra Jain™** trademark.
*   **Typography**: Uses **Space Grotesk** for high-impact headlines and **Inter** for readable enterprise data.
*   **Color Palette**: A "Dark-Mode First" aesthetic utilizing a deep charcoal background (`#0D0F16`) with Royal Blue (`#4169E1`) primary accents and Cyan (`#00BFFF`) secondary highlights.
*   **Aesthetic**: High-density enterprise grid with glassmorphism effects (backdrop-blur) on cards and sidebars.

## 2. Key Workspace Screens

### A. Role Selection (The Gateway)
The entry point of the application. It features six large, interactive tiles that allow users to select their persona. Each tile includes a unique icon and a description of the workspace's focus.

### B. Sales Representative Hub
A productivity-focused dashboard featuring real-time quota attainment bars, a "Stakeholder Ledger" with contact avatars, and a prioritized deal pipeline.

### C. Operational Velocity Pipeline
A kanban-style interface for deal management. Cards are color-coded by probability and show clear progression through stages from "Prospecting" to "Closed Won."

### D. Executive Leadership Hub
A high-level strategic snapshot featuring regional revenue areas charts and predictive modeling indicators. It serves as the jumping-off point for deep analytics.

## 4. Operational Control Centers
*   **Discount Approval Inbox**: A clean, list-based interface for managers to quickly process quote variations with one-click "Approve" or "Decline" actions.
*   **Product Bridge**: A collaborative feed mapping customer feedback directly to roadmap items using priority-coded badges.
